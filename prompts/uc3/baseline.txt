Design a valid ngspice netlist for an RC snubber circuit.

Specifications:
- DC supply voltage Vdc = 24V
- Load: series RL with Rload = 2 ohms, Lload = 10 mH
- Parasitic capacitance at switching node Cpar = 50 pF

**REFERENCE TOPOLOGY (follow this EXACT structure):**
* V1 in 0 DC 24V
* Rs in sw 0.1
* .model SWMOD SW(Ron=1m Roff=1Meg Vt=0.5 Vh=0)
* Vpulse ctrl 0 PULSE(0 1 0 1u 1u 1m 2m)
* S1 in sw ctrl 0 SWMOD
* Rload sw n1 2
* Lload n1 0 10mH
* Cpar sw 0 50pF
* Rsnub sw snub_mid 10
* Cs snub_mid 0 10nF
* .tran 0.1u 3m UIC
* .meas tran vpeak_switch MAX v(sw) FROM=0.9m TO=3m
* .meas tran snubber_energy INTEG (v(sw)-v(snub_mid))*i(Cs) FROM=0.9m TO=3m
* .meas tran settling_proxy MAX v(sw) FROM=1.5m TO=3m
* .end

**CRITICAL NODE NAMING:**
- DC supply positive: in
- Switching node: sw
- Load intermediate node: n1
- Snubber intermediate node: snub_mid
- Control node: ctrl
- Ground: 0

**CRITICAL - COPY THIS EXACTLY:**
.model SWMOD SW(Ron=1m Roff=1Meg Vt=0.5 Vh=0)
Vpulse ctrl 0 PULSE(0 1 0 1u 1u 1m 2m)
S1 in sw ctrl 0 SWMOD
.tran 0.1u 3m UIC

**MEASUREMENTS - COPY EXACTLY:**
.meas tran vpeak_switch MAX v(sw) FROM=0.9m TO=3m
.meas tran snubber_energy INTEG (v(sw)-v(snub_mid))*i(Cs) FROM=0.9m TO=3m
.meas tran settling_proxy MAX v(sw) FROM=1.5m TO=3m

**CRITICAL:**
- Snubber capacitor MUST be named 'Cs'
- Use (v(sw)-v(snub_mid))*i(Cs) NOT v(sw)*i(Cs)
- Include 'UIC' in .tran line

**FORBIDDEN:**
- .control blocks
- plot, print, wrdata
- active devices, diodes
- VDB, db, log, log10

Return ONLY the SPICE netlist. First line: * RC Snubber Circuit. Last line: .end
